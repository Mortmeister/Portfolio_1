# Portfolio Morten Lillehaug

## Overview

A single-page portfolio site built with HTML, CSS, and JavaScript, showcasing my three projects: Gamehub, Inspire Ink, and Community Science Museum.

## Live demo

https://portfolio-morten.netlify.app/

## Structure

- **Header strip**: name and social links (GitHub and LinkedIn), contact email
- **Carousel banner**: Project showcase with image, title, teaser, repo link, and live demo link for each project, navigable via arrows and dot buttons.

## Tech stacck

- HTML
- CSS
- JavaScript
